### Data Availability and Acknowledgements 
We thank the Mostafa Lab for coordinating and processing nasal swab samples for whole genome sequencing. Data are available via GISAID from The Johns Hopkins School of Medicine.

The Mostafa lab provides Influenza consensus data to the Pekosz Lab ahead of GISAID release. 

### Tanglegrams can be generated within each build using the format:
https://nextstrain.org/groups/PekoszLab-Public/{subtype}/{segment}:groups/PekoszLab-Public/{subtype}/{segment}

Example (h1:na):
https://nextstrain.org/groups/PekoszLab-Public/h1n1/ha:groups/PekoszLab-Public/h1n1/na

For more  details: [Nextstrain tanglegram documentation](https://docs.nextstrain.org/projects/auspice/en/stable/advanced-functionality/second-trees.html). Special thank you to Amary Fall, David Villafuerte, Madeline Yunker, and Omar Abdullah for their expertise in generating and sharing these sequences.
